# Moster Burger

Landing page de **Moster Burger** (hamburguesas para perros).

## Ver el proyecto en Visual Studio Code

1. Abre esta carpeta en VS Code:
   - `File > Open Folder...` y selecciona `Mosterburger`.
2. Instala la extensión oficial **JavaScript Debugger** (normalmente ya viene incluida).
3. Ejecuta la configuración de depuración:
   - Ve a `Run and Debug`.
   - Selecciona **Open Moster Burger in Browser**.
   - Presiona `F5`.
4. VS Code levantará un servidor local y abrirá la página en:
   - `http://localhost:4173`

## Alternativa rápida desde terminal integrada de VS Code

```bash
python3 -m http.server 4173
```

Luego abre en tu navegador: `http://localhost:4173`.
